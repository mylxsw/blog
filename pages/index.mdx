## 我是谁？

我是一名充满激情的编程爱好者、独立开发者以及软件架构师。我对人工智能充满浓厚的兴趣。利用空闲时间，我开发了一款整合多种主流语言模型的聊天 App（[AIdea](https://github.com/mylxsw/aidea)），该项目已经在 GitHub 开源。

我热衷于探索前沿技术，并将其灵活运用于实际项目之中。我坚信技术的火花能点燃行业变革，能够为行业发展和社会进步带来深远的影响。作为我的职业追求，我立志成为一名杰出的软件架构师，引领团队打造既高品质又高效能的软件系统，以不断推动行业向前发展。
